Linear Search

We implement linear search by traversing through every i element in a given list. We are checking one item at a time in sequential order. We will apply linear search to the following elements using an array:

int[] array = {1, 2, 3, 4, 5, 6, 6, 6, 7, 11, 5, 1};

Results

If an element is found, we will return True. If it is not found or the list is empty, we return False.

Search

Let us create a method in which we will take in a list of elements and the element we are searching for:

static bool Search(int[] arr, int n){
 
}

We ill iterate through every element of arr. If i is that element, we return True since it was found, else not:

for(int i = 0; i < arr.Length; i++){
  if(arr[i] == n){
   return true;
  }
}
return false;

Complexities

  • Best Case: O(1), element is first in list
  • Worst case: O(n), element is at n-1, or n
  • Average case: O(n), element is at n
  • Space complexity: O(1)
