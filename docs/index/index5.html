Sum of Two Values

Our approach to finding the sum of two values in a given list will consist of traversing through each element in the list starting at index = 0. We will implement this algorithm using a hash table. We will assume there can be repeated elements and contains either ℤ+ and or ℤ-. Assume we are given the following list of elements:

int[] array = {3, 4, 5, 4, 1, -2, 5};

Results

If the sum of two elements in the list are equal to value, then we return True, else False.

Search

We will first create a hash table. This will allow us to stores keys (indicies) and values (value of indicies). This will minimize our time complexity to O(n) at the expense of O(n) space.

Hashtable values = new Hashtable();

We will start with the first element and loop through n-1. We will then add to our hashtable i being the index of the current element in the loop, i.e. the key, and array[i] being the value of that index.

for(int i = 0; i < array.Length; i++){
  values.Add(i, array[i]); }

To find the element whos sum with N is the value passed in, we will use K - N. For example: if we pass in K = 13 and our current element is N = 6, then the only element that when added with 6 is 13 must be K - N or 13 - 6 = 7. Therefore, if our hashmap contains a value that is the difference of K and N, then ∃n1 ∈ array whos sum with ∃n2 ∈ array is the value paased in.

if(values.ContainsValue(value - array[i])){
  return true;
}

Complexities

  • Best Case: O(n)
  • Worst case: O(n), must our hashtable add and check every value
  • Average case: O(n), since our hash table only does O(1) lookup
  • Space complexity: O(n), since we are using a hash table at the exchange of space for speed
