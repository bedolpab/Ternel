Ternel.

Synopsis

Welcome to Ternel. This is a collection library of functions aimed at simplifying your C# development. These include basic functions such as converting an data type, input validation, animations, etc. While many are already implemented into the language, such as using LINQ, convenient methods such as printing an array without having to initialize a for loop are not.

We hope to expand this project to much more complex functions for different uses, such as in mathematics: Statistics, Probability, Calculus, etc.There are many broads concepts but the idea stands.

Example

Suppose you are working with arrays. To assure your code is working, you need to see all the elements of your array first.

This is very easy to achieve, though writing a function to do so in every program you make becomes annoying. To fix this, you might use one of our library's functions:

Use

int[] array = {10, 20, 30};
Ternel.DisplayArr(array);

Output may appear as so:

> 10
> 20
> 30

This library is in its beginning development stages, so code examples may vary.
