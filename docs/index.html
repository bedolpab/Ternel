Ternel.

Synopsis

Welcome to Ternel. This is a collection library of solutions and approaches to algorithmic and mathematical problems written strictly in C# or Pseudocode (for approaches for whom code has not yet been written).

We hope to expand this project to much more complex solutions and approaches for many subfields in the computer science and mathematics fields.
